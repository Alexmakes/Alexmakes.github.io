Project: In to the Ai-verse 

alex benett

as part of AutoCrat project i will need a way of generating text from a prompt and will all the hype of ChatGPT you think that would be a great contender for this and you be right, untill you remember EMF happens in a field in the middle of nowhere and internet will be slow enough that handling all the compute in the cloud is not an option so enter player 2, a dell 1U server with 128gb of ram. 

Adventure 

one long car journey and a lot of sweat and swearing as push a 1U server and monitor into the back of a hatchback in the middle of northwest

getting it setup on the desk before it goes in a rack, dahm these are loud, extra points for RGB. OS of choice for servers is Ubuntu server edition as its install is painless and has all the drivers for raid controllers and is possible to add open SHH from the install and no need to do it later.

All setup

OS installed and SHH working can sit back and enjoy getting an LLM installed.
